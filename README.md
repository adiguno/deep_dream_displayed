# deep_dream_displayed
Create DeepDream-like images and videos


## Getting Started with the Virtual Environment - Pipenv
Installing Pipenv:
```
pip install pipenv
```
Setting up pipenv:
```
pipenv install
```
Using pipenv:
```
pipenv shell
python [file_name]
```
